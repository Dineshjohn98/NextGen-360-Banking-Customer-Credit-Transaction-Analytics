# 🏦 NextGen 360 Banking – Customer, Credit & Transaction Analytics

## 📌 Project Overview

NextGen 360 Banking is an end-to-end Banking Data Analytics project developed using **Python (Google Colab)** and **Power BI Desktop**. The project analyzes customer profiles, banking transactions, loan performance, credit utilization, risk factors, and customer feedback to generate actionable business insights through interactive dashboards.

The project demonstrates the complete data analytics workflow—from data collection and cleaning to visualization and dashboard development—helping financial institutions make data-driven decisions.

---

## 🎯 Project Objectives

- Analyze customer, transaction, loan, and account data to identify banking trends and customer behavior.
- Evaluate loan approvals, repayment patterns, credit utilization, and financial risk.
- Detect anomalous transactions and classify customer risk levels.
- Build interactive Power BI dashboards for business intelligence and decision-making.
- Provide strategic recommendations to improve customer experience and banking operations.

# 📊 Project Workflow

## 1️⃣ Data Collection

- Collected a comprehensive banking dataset from GitHub.
- Verified dataset quality and structure.
- Checked missing values, duplicates, and inconsistent data types.

---

## 2️⃣ Data Cleaning & Preprocessing

- Handled missing values using appropriate imputation techniques.
- Replaced missing Email values with **"Unknown"**.
- Filled missing Account Balance values using the column mean.
- Converted Loan Issue Date into DateTime format.
- Removed inconsistencies and standardized data types.

---

# 📊 Key Insights

- Bank serves **5,000 customers**, with only **6% classified as high-risk**.
- Savings and Current accounts are almost equally distributed.
- Mortgage loans contribute the highest lending volume.
- Personal loans record the highest rejection rate.
- Q4 shows the highest loan disbursement across the year.
- San Jose, Indianapolis, and San Francisco have the largest customer base.
- MasterCard users accumulate the highest reward points.
- Customer suggestions slightly exceed complaints, indicating strong engagement.
